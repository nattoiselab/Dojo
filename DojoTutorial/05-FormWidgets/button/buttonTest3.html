Demo: A button using an icon (programmatic)

Below, we see dijit/form/Button in three forms: as a standard button, and using an icon with no label and another one with label. Be sure to open the console to see messages when clicking the buttons!

Explanation:

  • iconClass:' ': indicates what CSS class to use (to apply an image sprite)
  • showLabel: true: determines whether to show any text in the button
